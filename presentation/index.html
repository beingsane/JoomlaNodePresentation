What is Node?

It's JavaScript without a browser.

JavaScript, really?


  • Node uses Chrome's V8 JavaScript engine
  • Standard JavaScript + handful of globals
  • Each file has its own scope
  • Asynchronous JavaScript fully embraced

What can you build?


  • Server-side web applications
  • Command-line scripts
  • WebSocket applications
  • Flying robots with RESTful APIs
  • Anything you want

Node servers


  • Unlike PHP, they stay running
  • Designed to be non-blocking
  • Directly face the web

Modules


  • Self contained code
  • Core modules ands community modules
  • Manage project dependencies with NPM

Example - Taco.js


var http = require('http');

http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/html'});
  res.end("<h1>Here's a taco!</h1>");
}).listen(8888, '127.0.0.1');

console.log('Taco server ready at http://127.0.0.1:8888');

Joomla Node Module


  • Connect Joomla database and sessions to Node

var joomla = require('joomla');

joomla('/path/to/joomla/site');

joomla.db.query('SELECT * FROM #__users', function (rows) {
  rows.forEach(function  (row) {
    console.log({
      name: row.name,
      username: row.username,
      id: row.id
    });
  });
});

Socket.io + Joomla


  • Write WebSocket code quickly
  • Connect to Joomla using the Joomla module
  • Authenticate Joomla sessions through Node
  • Chat room!

Socket.io Backend

var joomla = require('joomla'),
  connect = require('connect'),
  io = require('socket.io');

joomla('/path/to/joomla/site');

var app = connect()
          .use(connect.static('public'))
          .listen(8000);

var chat_room = io.listen(app);

chat_room.sockets.on('connection', function (socket) {
  // your socket logic here
});

Socket Connection

// emitting a message to one user
socket.emit('error',
  {message: 'Please log in to join the chat'}

Socket Frontend

<!-- Include the Socket.io library in our template -->
<script src="http://localhost:8000/socket.io/socket.io.js">
</script>


// Connect to the websocket
var socket = io.connect('http://localhost:8000');


// send messages to the server
jQuery('#chat_box').keypress(function (event) {
  if (event.which == 13) {
    socket.emit('chat', {message: jQuery('#chat_box').val()});
    jQuery('#chat_box').val('');
  }
});
